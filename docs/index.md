# Nand2Tetris Notes

Personal notes and HDL solutions for the Nand2Tetris course, built bottom-up
from a single NAND gate to a working computer.

Each page covers the concept, truth table, HDL solution, and a video walkthrough.

| 2 | Boolean Arithmetic — adders, ALU |
| 3 | Sequential Logic — registers, RAM |
| 4 | Machine Language — Hack assembly |
| 5 | Computer Architecture — CPU, Memory |
| 6 | Assembler |
